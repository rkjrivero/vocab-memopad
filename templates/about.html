About VOCAB-MEMOPAD


VOCAB-MEMOPAD is a simple web app designed to quickly look up and save the translations of words/phrases and review them online.
Application uses Google Translate and supports translations across 15 different languages, but UI is currently only in English.
Currently supported languages are:
Currently Supported Languages: Translation API Used:
English Google Translate
Esperanto Google Translate
Japanese Google Translate
Latin Google Translate
Spanish Google Translate
German Google Translate
French Google Translate
Italian Google Translate
Russian Google Translate
Arabic Google Translate
Indonesian Google Translate
Malaysian Google Translate
Thai Google Translate
Vietnamese Google Translate
Tagalog Google Translate

Application UI currently only designed with desktop browsers in mind, and is not mobile-friendly.
DISCLAIMER: UI elements and underlying code borrow from CS50's Finance problem set.
