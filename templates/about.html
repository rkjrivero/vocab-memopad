VOCAB-MEMOPAD


About The App

VOCAB-MEMOPAD is a simple web app designed to quickly look up and save the translations of words/phrases and review them online, just like a small memopad to jot down vocabulary entries when studying a language. The application primarily uses Google Translate (via the googletrans python library) and supports translations across 15 different languages. Please note that the UI is currently only available in English.

Known Limitations

The application UI is currently only designed with desktop browsers in mind, and is not mobile-friendly.


Current Plans

A mobile-friendly UX is currently part of future development plans.
Similarly, there are some plans to incorporate other translation API's for certain languages, such as Jisho API for Japanese translations.
