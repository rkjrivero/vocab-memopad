VOCAB-MEMOPAD is a simple web app designed to quickly look up and save the translations of words/phrases and review them online.
Only three language translations are currently supported (English/Japanese/Esperanto), with the UI currently only in English.
Application UI is currently only designed with desktop browsers in mind, and is currently mobile-friendly.
DISCLAIMER: UI elements and underlying code borrow from CS50's Finance problem set.
